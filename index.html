Anomaly Detection with Deep Learning

Development of a novel Transformer-based architecture for anomaly detection in industrial sensor time series.

Python PyTorch Time Series
View Details

Detailed Description

This project focuses on creating robust Deep Learning models capable of identifying anomalous patterns in massive data streams from IoT sensors in industrial environments.

Methodology

  • Time series preprocessing using sliding window techniques and adaptive normalization.
  • Implementation of a modified Transformer architecture with sparse attention mechanisms to improve computational efficiency.
  • Unsupervised training using an error reconstruction approach.

Results

The proposed model outperformed traditional baselines (LSTM, Autoencoders) by 15% in terms of F1-Score on the public SWaT dataset.
